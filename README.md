Inspiration
- I wanted to create a macropad that allowed me to work more efficiently. I created a macropad using 6 switches, 1 rp2040, and 2 leds.

Challenges
- Believe it or not, this was my first time using Fusion 360 and KICad! I'm pretty proud of the final product. I had the most truble importing the PCB File into Fusion.

Specifications
BOM:

- 6x Cherry MX Switches
- 2x SK6812 MINI Leds
- 1x XIAO RP2040
- 4x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset

Others:

- KMK Firmware
- Heckerpad_luke_case.stl
- Top_Heckpad_Luke.stl
